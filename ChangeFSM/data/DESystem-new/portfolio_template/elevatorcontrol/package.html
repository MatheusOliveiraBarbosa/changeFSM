This page lists and describes the java files in your elevatorcontrol package.  Update this document to reflect your implementation and add your java files to the elevatorcontrol folder.

Note:  In order for your portfolio to be complete, the code files in the elevatorcontrol folder must compiled and run if copied into the elevatorcontrol package (overwriting any duplicates) of a clean version of the latest required simulator release.

Controller files

  • DoorControl.java - Controller that implements the behavior for the Doors Motors in the elevator system.
  • HallButtonControl.java - Controller that implements the behavior for the HallLights and HallCalls in the elevator system.
  • CarButtonControl.java - Controller that implements the behavior for the CarLights and CarCalls in the elevator system.
  • DriveControl.java - Controller that implements the behavior for the Drive in the elevator system.
  • Dispatcher.java - Controller that implements the behavior of the dispatcher that sets the target of the elevator in the elevator system.
  • CarPositionControl.java - Controller that is used to display the current position of the elevator in terms of floors.
  • LanternControl.java - Controller that is used to light up the car lanterns in the elevator to display the next direction of the elevator system.

Translator files

Links and descriptions to all message translators used by the controllers.
  • DesiredFloorCanPayloadTranslator.java - Translator for the mDesiredFloor[f,b,d] message. Packages the three fields into a bit level representation.
  • DoorMotorCommandCanPayloadTranslator.java - Translator for the DoorMotor Commands.
  • DriveCommandCanPayloadTranslator.java - Translator for Drive Commands.
  • DriveSpeedCanPayloadTranslator.java - Translator for the DriveSpeed message.
  • DesiredDwellCanPayloadTranslator.java - Translator for the DesiredDwell message.
  • BitCanPayloadTranslator.java - Translator for Sending one byte messages.
  • IntCanPayloadTranslator.java - Translator for sending Integer type messages.

Utility files

  • Proj7RuntimeMonitor.java - RunTime Monitor for Project 7 Working basic elevator.
  • Proj11RuntimeMonitor.java - RunTime Monitor for Project 11 Working advanced elevator.
  • Utility.java - Utility class that contains helper functions
  • MessageDictionary.java - Message Dictionary that contains message periods and CAN IDs of messages.
